# Agent Instructions

This folder holds `skill-creator` — a skill for creating, testing, and iteratively improving other skills. Work here is about improving skill-creator itself, not about using it to build some other skill. Example prompts the user gives while discussing it are eval inputs for skill-creator's workflow, not real work.

Read `CLAUDE.md` in this folder for the full instructions: eval mode vs. production mode, the deliberate removal of automated grading (do not reintroduce it), and the known failure modes to watch for.
